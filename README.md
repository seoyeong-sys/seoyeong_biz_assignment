# 🧸 몽글몽글 감성 다이어리 달력 (Emotional Diary Calendar)

사용자가 한 달 동안의 일정, 감정, 날씨를 기록하고 자유롭게 꾸밀 수 있는 디지털 달력 서비스입니다. 파스텔톤 다이어리 감성의 인터페이스와 풍부한 커스터마이징, 통계 분석, 그리고 작성된 달력을 이미지로 내보내는 기능까지 모두 제공하는 Streamlit 단일 파일 웹 애플리케이션입니다.

---

## 📂 프로젝트 구조

```text
streamlit-diary-calendar/
├── app.py                  # Streamlit 메인 애플리케이션 코드
├── requirements.txt        # 라이브러리 의존성 목록
├── README.md               # 본 설명서 파일
└── data/                   # 달력 데이터 저장소 (앱 실행 시 자동 생성됨)
    ├── cal_<calendar_id>.json # 개별 달력 데이터 파일들
    └── last_active.json    # 마지막으로 수정한 달력 ID 기록
```

---

## 🚀 로컬 환경 실행 방법

이 프로젝트를 로컬 개발 환경에서 구동하는 방법입니다.

### 1. 패키지 설치
터미널을 열고 프로젝트 디렉토리로 이동한 뒤, 아래 명령어를 실행하여 필요한 의존성 라이브러리를 설치합니다.
```bash
pip install -r requirements.txt
```

### 2. Streamlit 실행
설치가 완료되면 다음 명령어로 애플리케이션을 구동할 수 있습니다.
```bash
streamlit run app.py
```
브라우저에서 `http://localhost:8501` 주소로 자동으로 웹앱이 활성화됩니다.

## 🌐 Streamlit Cloud 배포 방법

이 프로젝트는 단일 파일 기반으로 제작되어 Streamlit Cloud에 즉시 무료로 배포가 가능합니다.

### 1. GitHub 저장소 생성
1. 본인의 GitHub에 로그인하고 새로운 Repository(저장소)를 생성합니다.
2. 로컬에 생성한 `app.py`, `requirements.txt`, `README.md` 파일을 해당 GitHub 저장소에 푸시(Push)합니다. (`data` 폴더는 푸시하지 않아도 됩니다.)

### 2. Streamlit Cloud 연동 및 배포
1. [Streamlit Community Cloud](https://share.streamlit.io/)에 접속하여 가입 및 로그인을 진행합니다.
2. 우측 상단의 **[Create app]** 버튼을 클릭합니다.
3. 생성한 GitHub 저장소(Repository), 브랜치(Branch - 보통 `main` 또는 `master`), 그리고 메인 파일 경로(`app.py`)를 선택합니다.
4. **[Deploy!]** 버튼을 누르면 클라우드 컨테이너 빌드가 진행되고 배포가 완료됩니다!

## ✨ 제공 기능 설명

1. **다중 달력 관리**:
   - 사이드바를 통해 여러 종류의 달력을 생성하고 불러올 수 있습니다. (예: "여행 계획 달력", "2026년 7월 일정" 등)
   - 달력을 완전히 삭제하는 기능을 함께 제공합니다.
2. **다크 모드**:
   - 사이드바의 토글 스위치 하나로 다크 모드를 활성화하여 밤에도 편리하게 눈 피로 없이 다이어리를 쓸 수 있습니다.
3. **이모티콘 및 스티커 데코레이션**:
   - 각 날짜별로 감정(10종)과 날씨(8종) 이모티콘을 쉽고 간단하게 등록할 수 있습니다.
   - 다양한 스티커를 날짜별로 중복해서 붙여 풍성하게 꾸밀 수 있습니다.
4. **컬러 커스터마이징**:
   - 파스텔 핑크, 블루, 민트, 라벤더 등 완성도 높은 기본 테마를 원클릭으로 변경할 수 있습니다.
   - 사용자 지정을 선택해 컬러 피커로 본인의 다이어리 파스텔 색 조합을 직접 자유롭게 커스터마이징할 수 있습니다.
5. **통계 분석 📊**:
   - 한 달 동안 다이어리에 기록된 감정 분포와 날씨 빈도를 시각화한 감성 차트를 분석하여 월별 삶의 패턴을 진단해 줍니다.
6. **이미지 내보내기 🎨**:
   - Matplotlib을 이용해 화면상에 꾸민 이모티콘과 스티커, 메모를 완벽하게 반영한 단일 고화질 이미지(PNG)를 자동 생성하고 다운로드할 수 있습니다.
7. **JSON 백업 백업 및 복원**:
   - 데이터를 수동으로 내보내어 소장하거나, 다른 기기에서 가져오기 기능을 통해 그대로 완벽하게 복원하여 이어 쓸 수 있습니다.
